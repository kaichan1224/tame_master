# garimuki_scripts
- 動物と戯れながら主人公をムキムキにさせる位置情報ゲーム「Garimuki」のソースコード

# 使用技術
- UniRx(ReactiveProperty)
- UniTask
- MapboxAPI

# ゲーム内容
- 歩くとキャラクターが進化するぞ!
- 動物全部捕まえてモンスターマスターを目指せ!!
